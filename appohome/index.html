Appohome

En desarrollo

Se trata de una aplicación encargada por un cliente, cuyo objetivo es conectar a personas que ofrecen servicios de cuidado personal, como cortes de cabello, barbería, manicura, pedicura, entre otros, con clientes que desean recibir dichos servicios desde la comodidad de sus hogares.

La aplicación ofrece una amplia gama de servicios, desde la búsqueda de profesionales hasta la reserva de citas y la automatización de pagos.

Programada en React Native, esta app está disponible tanto para dispositivos Android como iOS. Para el backend, se han utilizado diversas tecnologías, incluyendo C#, mientras que para la base de datos se ha empleado GraphQL y PostgreSQL.
